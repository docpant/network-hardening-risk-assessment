# Network Hardening Risk Assessment

## Overview

This project analyzes security vulnerabilities identified within a social media organization after a major data breach exposed customer information.

The assessment focuses on identifying weak security practices, evaluating network vulnerabilities, and recommending network hardening methods to improve overall security posture.

## Objective

- Identify major security vulnerabilities
- Recommend network hardening techniques
- Analyze security risks
- Improve authentication and access controls
- Strengthen network protection measures

## Vulnerabilities Identified

- Employees shared passwords
- Default administrator password was enabled
- Firewall rules were not properly configured
- Multifactor authentication (MFA) was not implemented

## Hardening Methods Recommended

- Strong password policies
- Multifactor authentication (MFA)
- Firewall rule configuration
- Account access monitoring
- Security awareness training

## Project Structure

- risk-assessment.md
- network-hardening-methods.md
- security-recommendations.md
- findings.md

## Outcome

The assessment identified multiple weaknesses that increased the risk of future breaches. Recommended hardening measures were provided to strengthen authentication, network protection, and overall security management.
