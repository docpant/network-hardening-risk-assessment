# Security Risk Assessment

## Incident Summary

A social media organization experienced a major data breach that exposed customer personal information, including names and addresses.

The investigation identified several security vulnerabilities that increased the likelihood of unauthorized access and future attacks.

## Vulnerabilities Identified

### Password Sharing
Employees shared passwords across accounts, increasing the risk of unauthorized access and credential compromise.

### Default Administrator Password
The database administrator account still used the default password, making it vulnerable to brute force attacks and unauthorized access.

### Weak Firewall Configuration
Firewall rules were not configured to properly filter inbound and outbound traffic, increasing exposure to malicious traffic.

### Missing Multifactor Authentication
The organization did not implement MFA, reducing account security and increasing the risk of credential compromise.

## Risk Impact

If these vulnerabilities remain unresolved, the organization could experience:
- additional data breaches
- unauthorized system access
- malware infections
- credential theft
- operational disruption
